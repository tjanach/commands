# Python 2 scripting

+ check python version
```sh
$ python -V
```
+ enter python interactive shell
```sh
$ python
```
+ close python interactive shell
```sh
ctrl+d
```
+ simple input script
```python
#!/usr/bin/python

print ("Please enter your name: ")
input = raw_input()
print ("Hello "+input)
```

```python
#!/usr/bin/python
if condition:
    code
elif condition:
    code
else:
    code
```
